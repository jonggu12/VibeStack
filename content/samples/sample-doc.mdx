# Next.js 14 시작하기

Next.js 14는 React 기반의 풀스택 프레임워크입니다. 이 문서에서는 Next.js 14의 주요 기능과 프로젝트 시작 방법을 알아봅니다.

## 주요 기능

Next.js 14의 주요 기능은 다음과 같습니다:

- **App Router**: 새로운 라우팅 시스템
- **Server Components**: 서버에서 렌더링되는 컴포넌트
- **Server Actions**: 서버에서 실행되는 함수
- **Streaming**: 점진적 렌더링

<Callout type="info">
Next.js 14는 React 18의 최신 기능을 완전히 지원합니다.
</Callout>

## 프로젝트 생성

새 Next.js 프로젝트를 생성하려면 다음 명령어를 실행하세요:

```bash
npx create-next-app@latest my-app
```

설치 중 몇 가지 옵션을 선택할 수 있습니다:

```
✔ Would you like to use TypeScript? Yes
✔ Would you like to use ESLint? Yes
✔ Would you like to use Tailwind CSS? Yes
✔ Would you like to use `src/` directory? No
✔ Would you like to use App Router? Yes
```

<Callout type="tip" title="권장 설정">
TypeScript와 App Router를 사용하는 것을 권장합니다. 더 나은 개발 경험을 제공합니다.
</Callout>

## 개발 서버 실행

프로젝트 디렉토리로 이동하여 개발 서버를 실행합니다:

```bash
cd my-app
npm run dev
```

브라우저에서 `http://localhost:3000`을 열면 Next.js 앱을 확인할 수 있습니다.

<Checkpoint id="setup-complete" title="프로젝트 설정 완료">
개발 서버가 정상적으로 실행되면 이 체크포인트를 완료하세요.
</Checkpoint>
