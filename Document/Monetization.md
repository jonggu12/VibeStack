

## 6. 수익 모델 (Monetization)

### 6.1 수익 전략 개요

**핵심 철학: "사용자 선택권 극대화"**

```
3가지 수익 옵션: 
├─ 🆓 Free: 기본 학습 (무료) 
├─ 💰 Pay-per-Content: 필요한 것만 구매 
└─ ✨ Pro Subscription: 올인원 무제한

→ "보고 싶은 것만 사거나, 구독하거나"

````

**전략적 장점:**
- ✅ 진입 장벽 완전 제거 (무료로 시작)
- ✅ 수익 다각화 (단품 + 구독 + 제휴)
- ✅ 자연스러운 업셀 (강요 없음)
- ✅ 콘텐츠 가치 실시간 검증
- ✅ 리스크 분산 (구독 실패해도 수익)

### 6.4 전환 전략 (Conversion Strategy)

#### A. 무료 → 단품 전환

```yaml
트리거 포인트:
  1. 튜토리얼 완성 후:
     "🎉 Todo 앱 완성!
      이제 결제 기능 추가해볼까요?
      
      💰 Stripe 튜토리얼 - $12
      [미리보기] [구매하기]"
  
  2. 검색 시 페이월:
     "🔒 이 문서는 프리미엄이에요
      
      'Stripe Webhook 가이드' - $5
      또는 Pro로 전체 잠금 해제
      
      [$5 구매] [Pro 알아보기]"
  
  3. 에러 진단 제한:
     "⚠️ 이번 달 무료 진단 모두 사용
      
      • 계속 사용: Pro 구독
      • 이번만: 단품 구매
      
      [Pro 가입] [단품 보기]"

심리 기법:
  - FOMO: "892명이 이미 완성했어요"
  - 사회적 증거: ⭐ 4.8/5.0
  - 즉시 가치: "2시간 만에 완성"
  - 저위험: "30일 환불 보장"
```
